# Task Automation Dashboard

A small internal-style dashboard for triggering automation tasks and tracking execution history.
Built as a portfolio project to demonstrate full-stack TypeScript: React frontend + Express API + Prisma + SQLite.

## Features (current)
- List available task definitions
- Trigger a task run
- Track run status (queued → running → success/failed)
- View run details and logs in the UI

## Tech stack
- Web: React + Vite + TypeScript
- API: Node.js + Express + TypeScript
- DB: Prisma + SQLite

## Running locally

### Prereqs
- Node.js (your standard dev install)
- npm

### Install
From the repo root:

```bash
npm install
